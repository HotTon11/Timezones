TIMEZONES

Timezone calculations within the built-in Javascript framework. Returns local time of the 3 places I've lived + GMT.

Greenwich (GMT)

New York (EST)

Dallas (CST)

Singapore (SNST)

There was more to this that I thought. Here's a couple important takeaways:
  1. My first time creating a "timer" using setTimeout(). In this case the timer function (set to 1000ms, or 1 sec) goes inside the main function returnTime(), which is called "on-load" in the body element.
  2. Javascript has built-in time calculations, but render different values depending on which method you are using. For example, getTimezoneOffset() returns minutes whereas getTime() returns milliseconds.
  3. UTC functions are built into Javascript. I didn't use any in the application, but depending on how you need manipulate and/or display the date, that functionality could be useful.
